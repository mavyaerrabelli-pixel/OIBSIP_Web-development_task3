# Temperature Converter

## Objective

To develop a temperature converter web application using HTML, CSS, and JavaScript that converts temperature values between Celsius, Fahrenheit, and Kelvin.

## Steps

1. Designed the user interface using HTML for structure
2. Styled the application using CSS for layout and visual clarity
3. Implemented temperature conversion logic using JavaScript
4. Added input validation and tested the application functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Purpose

This project was developed to practice JavaScript fundamentals and demonstrate the integration of HTML, CSS, and JavaScript in building an interactive web application.
